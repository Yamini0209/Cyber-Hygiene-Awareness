# 🌐 Cyber Hygiene Awareness Website with AI Content Generator

## 📌 Project Overview

This project is a **Cyber Hygiene Awareness Website** designed to educate users about safe online practices.
It also includes a simple **AI-based content generator** that provides cybersecurity tips based on user input.

---

## 🎯 Objectives

* To spread awareness about **cyber hygiene practices**
* To help users understand **common cyber threats**
* To provide **interactive learning** using a quiz
* To simulate **AI-generated cybersecurity content**

---

## 🛠️ Technologies Used

* **HTML** – Structure of the website
* **CSS** – Styling and layout
* **JavaScript** – Interactivity and logic

---

## ✨ Features

* 🏠 Home section with introduction
* 🔐 Cyber safety tips
* ⚠️ Common cyber threats
* 🤖 AI Content Generator (keyword-based logic)
* 🧠 Cyber awareness quiz
* 🔄 Dynamic page navigation (Single Page Application)

---

## 🤖 AI Content Generator

The AI generator is implemented using **JavaScript logic**.
It analyzes the user input (topic) and generates relevant cybersecurity advice.

Example:

* Input: *password*
* Output: *Use strong passwords with letters, numbers, and symbols.*

---

index.html   → Main file (HTML + CSS + JS combined)

1. Download or clone the repository
